# HR Analysis Dashboard - Attrition Analysis

## Overview

This project analyzes employee attrition using Power BI to identify workforce trends and support HR decision-making.

## Dashboard Features

- Total Employees KPI
- Total Attrition KPI
- Attrition Rate
- Average Age
- Average Salary
- Average Tenure
- Attrition by Department
- Attrition by Gender
- Attrition by Education
- Attrition by Salary
- Attrition by Travel
- Attrition by Marital Status
- Interactive Filters

## Tools Used

- Power BI
- Microsoft Excel

## Dataset

HR Analytics Dataset - https://www.kaggle.com/datasets/anshika2301/hr-analytics-dataset

## Key Insights

- Calculated employee attrition rate.
- Compared attrition across departments, education levels, salary groups, and travel frequency.
- Built interactive visualizations to support HR decision-making.
